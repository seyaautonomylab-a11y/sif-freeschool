A01 チームビルディング +

チームの目標を共有し、役割分担と信頼関係を築く力。異なる世代・個性が混ざるSIFの場で、自然と身につきます。

A02 周囲を巻き込むコミュニケーション +

自分の考えを伝え、他者を動機づけ、協力を引き出す力。世代を超えたゲームの中で、自然と磨かれます。

A03 D&I（多様性＆包摂） +

異なる世代・背景・価値観を持つ人と共に楽しむ力。小学生が大人と対等に意見を言い合う環境が、この感覚を育てます。

A04 エンパワーメント +

他者の強みを引き出し、自信と主体性を育む力。「あなたの発想はすごい」と認め合う場が、子どもの自己効力感を高めます。

A05 ステークホルダーマネジメント +

関係者それぞれの立場・関心を理解し、合意を形成する力。大人・高校生・子どもが混在する場で実践的に学びます。

A06 ファシリテーション +

場を整え、意見を引き出し、議論を前進させる力（傾聴・質問・論点の構造化）。振り返りの場で毎回実践します。

A07 心理的安全性 +

「失敗しても大丈夫」と思える環境をつくり、挑戦を促す力。「間違えた人が一番考えた」を合言葉に、安心の場を設計します。

A08 組織・チームの類型 +

チームの状態（発展段階・タイプ）を見極め、状況に合ったアプローチを選ぶ力。多様なゲーム形式の中で体感します。

A09 未来思考 +

変化を予測し、今すべきことを逆算して考える力。「このゲームを勝つには？」という問いが、未来から逆算する習慣を育てます。

A10 ストーリーテリング +

自分の考えや経験を、相手の心に届く「物語」として伝える力。振り返りで「今日の成長を言葉にする」練習が積み重なります。

A11 ビジョナリーリーダーシップ +

先を見据えた方向性を示し、周囲を引っ張っていく力。子どもが大人を指導する「逆転状況」が、リーダー性を引き出します。

A12 デザイン思考 +

相手の立場に立って課題を発見し、創造的な解決策を生み出す力。「相手はどう感じているか？」を常に問う遊びの設計に組み込まれています。

A13 水平思考 +

固定観念を外し、予想外の視点から問題を解く力。「ウミガメのスープ」など、常識を覆すゲームで自然と鍛えられます。

A14 仮説思考 +

情報が不完全な状態でも仮説を立て、検証しながら前に進む力。限られた情報でゲームを進める体験が、この力を育てます。

A15 フェイルファースト +

小さく試して失敗から学び、素早く改善していく力。ゲームの中での失敗を「発見」として喜べる空気が、チャレンジ精神を育てます。

A16 批判的思考・論理的思考 +

情報の根拠を問い、筋道を立てて考え、正確に判断する力。「なぜそう思う？」を問い続ける講師のかかわりが、この習慣をつくります。

A17 システム思考 +

物事のつながりや全体像を把握し、本質的な原因にアプローチする力。ゲームの「なぜ負けたか」を構造で考える振り返りで育まれます。

A18 アジリティ +

変化に素早く対応し、柔軟に行動を切り替える力。毎回変わるゲームと相手に合わせて戦略を変える経験が、この力を磨きます。

A19 OODAループ +

観察→判断→意思決定→行動のサイクルを高速で回す力。テンポの速いゲームの中で、無意識にこのループを実践していきます。

A20 リーン思考 +

無駄を省き、本質的な価値に集中して成果を最大化する力。「何が一番大事か」を絞り込んで行動する習慣が、遊びの中で自然と育ちます。

FAQ — よくある質問

保護者の方からよくいただく質問

Q 参加頻度・時間帯は？ +
毎週火曜日 17:15〜18:45（90分）に開催しています。月4回の参加が基本です。都合が悪い週は事前にご連絡ください。
Q 人見知りの子でも大丈夫ですか？ +
むしろ人見知りの子ほど変化が大きいです。ゲームを通じた自然な関わりから始まるため、無理に話さなくても溶け込める環境です。
Q 費用はいくらかかりますか？ +
最初の2回は無料で体験いただけます。体験後にご入会いただいた場合は、参加チケット4回分で8,000円です。参加チケットは、フリースクールの開催日（毎週火曜日）にお使いいただけるチケットです。
Q 何歳から参加できますか？ +
小学3年生〜6年生を目安にしています。ゲームのルール理解や他の参加者とのコミュニケーションを楽しめる年齢として、この学年を推奨しています。また、お子さんと一緒に保護者の方が参加されるケースもあります。ご不明な点はお気軽にお問い合わせください。
Q 学校に通いながら参加できますか？ +
はい、通常の学校と並行して参加されているお子さんがほとんどです。学校の学びを補完・強化する形でご活用いただけます。
Q 体験後に必ず入会しなければなりませんか？ +
いいえ。無料体験（2回まで）後に、ご家庭でゆっくりご判断いただけます。入会の強制は一切ありません。
